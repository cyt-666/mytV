# MyTV

MyTV 是一款基于 Tauri (Rust + Vue 3) 开发的第三方 Trakt 客户端。它旨在提供流畅、现代且功能丰富的影视追踪体验。

## 主要功能

### 1. 智能搜索与探索
*   **搜索历史**：实现本地搜索历史记录，并支持持久化存储，方便快速找回之前的搜索内容。
*   **热门期待榜**：集成 Trakt Anticipated API，为您呈现当前全球范围内最受期待的电影和电视剧。
*   **多维筛选**：支持在搜索结果中按类型（电影/剧集）、年份、流派进行实时过滤。

### 2. 待看进度追踪 (Up Next)
*   **全新列表布局**：重新设计的“待看”页面采用列表布局，优化了信息密度，更适合追踪多部剧集的进度。
*   **进度可视化**：增强的进度条设计（采用蓝紫渐变效果），直观显示您的追剧进度百分比及具体集数。
*   **快速续播**：直接展示下一集的标题和编号，点击即可快速跳转至详情。

### 3. 个性化推荐
*   **定制化首页**：登录后，首页将调用 Trakt 推荐算法，根据您的观看喜好为您量身定制电影和剧集列表。
*   **热门趋势**：针对游客用户，系统会自动降级展示当前最热门的流行内容，确保首页内容始终丰富。

### 4. 剧集日历
*   **专属日历**：基于您的观看清单（Watchlist）和正在追的剧集，自动生成个性化的播出时间表。
*   **时间线视图**：清晰的日期分组和时间点标注，让您对未来一周的剧集更新了如指掌。

### 5. 系统设置与调试
*   **日志级别配置**：提供图形化界面配置应用日志级别（Debug/Info/Warn/Error），方便开发者和高级用户排查问题。
*   **配置持久化**：应用配置自动同步至本地文件，重启后依然保持您的个性化设置。

### 6. 极致交互体验
*   **全局导航优化**：顶部标题栏集成全局返回按钮，并支持 `Alt + ←` 快捷键，操作更符合原生应用习惯。
*   **自动回顶**：页面跳转时自动滚动至顶部，并在返回时智能恢复之前的滚动位置。
*   **精致 UI 细节**：基于 Arco Design 深度定制，包含沉浸式轮播图、优化的加载状态、空状态提示等。

## 开发环境配置

### 前提条件
*   [Node.js](https://nodejs.org/) (建议最新 LTS 版本)
*   [Rust](https://www.rust-lang.org/) 开发环境
*   [Yarn](https://yarnpkg.com/)

### 安装依赖
```bash
yarn install
```

### 启动开发环境
```bash
# 启动 Tauri 开发模式（包含前端热重载）
yarn tauri dev
```

### 构建正式版本
```bash
yarn tauri build
```

## 技术栈
*   **后端**: [Rust](https://www.rust-lang.org/), [Tauri](https://tauri.app/)
*   **前端**: [Vue 3](https://vuejs.org/) (Composition API), [TypeScript](https://www.typescriptlang.org/)
*   **UI 框架**: [Arco Design Vue](https://arco.design/vue/)
*   **工具库**: Vue Router, [ve-plus](https://github.com/v-easy/ve-plus)
